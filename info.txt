$$ npm init - инициализация проекта, ответы на баз.вопросы
в корне back, папка clien front
++ app.js баз.ф.настр.сервера
$$ npm i express mongose - устан.баз.завис.прилож. express (фреймворк для nodejs), mongoose (соед. с базой и раб.с mongodb)
$$ npm i -D nodemon concurrently - пакеты для разработки. автообнов.серв. и одновремен.раб. back и front 
## "start": "node app.js", "server": "nodemon app.js" - правки script в packege.json. start - автоматизация прилож. с запуском, server - запуск backend (разраб)
## в app.js - созд.баз.прилож.на express. получ.ч/з require,присвойка сервера к перемен.(напр.app),вызов слушат.(listen) с портом и fn,
$$ npm i config - устан.отд.конфиг настроек. 
++ config/default.json - конст.для разраб. в json
пропис.настр. в default.json,подкл.к app.js,использ.
